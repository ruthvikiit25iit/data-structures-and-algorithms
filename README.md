The goal of this SoS project is to build strong foundations in Data Structures and Algorithms using C++ and gradually move towards solving standard interview-level and competitive programming problems.

The primary focus of this project will be:

* Understanding core programming concepts in C++
* Learning fundamental data structures and algorithms
* Developing problem-solving skills

* Building algorithmic intuition through CLRS and lectures

The project is planned over 8 weeks with a gradual increase in difficulty and problem-solving exposure.
